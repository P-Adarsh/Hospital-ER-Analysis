# 🏥 Hospital Emergency Room Analysis

## 📌 Project Overview

This project analyzes Emergency Room patient data to evaluate hospital performance, patient flow, and service quality.

## 📊 Key Objectives

* Data Cleaning and Preparation
* Feature Engineering (Date, Age Groups)
* Outlier Detection (IQR and Z-Score)
* KPI Development
* Correlation Analysis
* Dashboard Creation

## 📈 Key Insights

* Average Patient Wait Time: ~35 minutes
* Admission Rate: ~50%
* Longer wait times negatively impact patient satisfaction
* Certain departments show higher wait times and lower satisfaction

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Advanced Excel Functions

## 📂 Files Included

* Task 1: Data Cleaning & Preparation
* Task 2: Analysis, KPIs & Dashboard

## 📌 Conclusion

The project highlights how operational efficiency directly impacts patient satisfaction and hospital performance.
